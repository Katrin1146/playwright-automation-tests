# Demoblaze Playwright Tests

End-to-end and API test automation project using Playwright and TypeScript.

## Covered Scenarios

### UI Tests
- User signup
- User login
- Add product to cart
- Remove product from cart
- Place order

### API Tests
- GET products
- POST login negative

## Tech Stack
- Playwright
- TypeScript
- Page Object Model (POM)

## Run tests

npm install  
npx playwright test
